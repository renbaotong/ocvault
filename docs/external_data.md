# 外部数据源对接评估

## Wind（万得）

### 基本信息
- **运营方**：上海万得信息技术有限公司
- **定位**：金融机构行业标准数据终端

### 费用（2025-2026）
| 用户类型 | 费用 |
|---------|------|
| 机构基础终端 | ~¥39,800/年 |
| API 接口 | 数万元~/年 |
| 个人用户 | 不推荐（成本过高） |

### 数据覆盖
- ✅ 债券发行数据
- ✅ 宏观数据
- ✅ 行业数据
- ✅ 财务指标
- ✅ 地方政府财政数据

### 对接方式
- 需联系销售报价（一客一价）
- 机构客户可申请 API 权限
- Python/Matlab/R/Excel 接口

### 推荐指数：⭐⭐⭐
**适用场景**：有机构预算的团队

## 企业预警通

### 基本信息
- **运营方**：上海大智慧财汇数据科技有限公司
- **定位**：企业信用与债券信息平台

### 费用
- 未公开报价，需商务洽谈
- 主要面向机构定制

### 数据覆盖
- ✅ 企业工商信息
- ✅ 债券融资信息
- ✅ 高管/实控人
- ✅ 经营风险

### 对接方式
- 无公开 API 文档
- 需申请试用后合作

### 推荐指数：⭐⭐⭐
**适用场景**：需要企业关联关系数据

## Tushare 数据接口测试（2026-04-11）

### 已验证可用接口（VIP>5000 积分）

| 接口类别 | 接口名 | 数据内容 | 可用状态 |
|---------|--------|---------|---------|
| **价格指数** | `cn_cpi` | CPI 居民消费价格指数 | ✅ 可用 |
| | `cn_ppi` | PPI 工业生产者出厂价格指数 | ✅ 可用 |
| **景气度** | `cn_pmi` | PMI 采购经理指数 | ✅ 可用 |
| **国民经济** | `cn_gdp` | GDP 国内生产总值 | ✅ 可用 |
| **货币供应** | `cn_m` | M0/M1/M2 货币供应量 | ✅ 可用 |
| **利率数据** | `shibor` | Shibor 银行间拆借利率 | ✅ 可用 |
| | `shibor_lpr` | LPR 贷款市场报价利率 | ✅ 可用 |
| | `yc_cb` | 中债国债收益率曲线 | ✅ 可用 |
| **社会融资** | `sf_month` | 社融增量月度数据 | ✅ 可用 |
| **债券市场** | `cb_basic` | 可转债基础信息 | ✅ 可用 |
| | `cb_daily` | 可转债行情 | ✅ 可用 |
| | `repo_daily` | 债券回购日行情 | ✅ 可用 |
| | `bond_blk_detail` | 债券大宗交易明细 | ✅ 可用 |

### 不可用接口

| 接口名 | 错误信息 | 可能原因 |
|--------|---------|---------|
| `bond_basic` | 请指定正确的接口名 | 接口名称变更或需要更高权限 |
| `bond_daily` | 请指定正确的接口名 | 同上 |

### 数据更新方法

```bash
# 设置环境变量（如未设置）
export TUSHARE_TOKEN=your_token

# 运行同步脚本
python3 scripts/tushare_data_sync.py
```

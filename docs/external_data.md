# 外部数据源对接评估

## Wind（万得）

### 基本信息
- **运营方**：上海万得信息技术有限公司
- **定位**：金融机构行业标准数据终端

### 费用（2025-2026）
| 用户类型 | 费用 |
|---------|------|
| 机构基础终端 | ~¥39,800/年 |
| API 接口 | 数万元~/年 |
| 个人用户 | 不推荐（成本过高） |

### 数据覆盖
- ✅ 债券发行数据
- ✅ 宏观数据
- ✅ 行业数据
- ✅ 财务指标
- ✅ 地方政府财政数据

### 对接方式
- 需联系销售报价（一客一价）
- 机构客户可申请 API 权限
- Python/Matlab/R/Excel 接口

### 推荐指数：⭐⭐⭐
**适用场景**：有机构预算的团队

## 企业预警通

### 基本信息
- **运营方**：上海大智慧财汇数据科技有限公司
- **定位**：企业信用与债券信息平台

### 费用
- 未公开报价，需商务洽谈
- 主要面向机构定制

### 数据覆盖
- ✅ 企业工商信息
- ✅ 债券融资信息
- ✅ 高管/实控人
- ✅ 经营风险

### 对接方式
- 无公开 API 文档
- 需申请试用后合作

### 推荐指数：⭐⭐⭐
**适用场景**：需要企业关联关系数据

## 替代方案

### Tushare
| 项目 | 详情 |
|------|------|
| 费用 | ~¥2,000/年 |
| 数据 | 宏观、债券、股票、基金 |
| 接口 | Python API |
| 推荐 | ⭐⭐⭐⭐⭐ 个人/小团队首选 |

### AkShare
| 项目 | 详情 |
|------|------|
| 费用 | 免费 |
| 数据 | 有限（开源社区维护） |
| 接口 | Python |
| 推荐 | ⭐⭐⭐ 学习研究 |

### Choice（东方财富）
| 项目 | 详情 |
|------|------|
| 费用 | 机构版（低于 Wind） |
| 数据 | 债券数据较全 |
| 接口 | Python/Excel |
| 推荐 | ⭐⭐⭐⭐ Wind 替代方案 |

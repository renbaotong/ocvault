# 公司债募集说明书知识库 - 使用指南

## 快速开始

### 1. 处理新募集说明书

```bash
# 将 PDF 放入 raw/ 目录
cp new_prospectus.pdf raw/

# 运行提取脚本
python3 scripts/extract_prospectus.py
```

### 2. 在 Obsidian 中打开

```bash
# 用 Obsidian 打开 knowledge 目录
open -a Obsidian knowledge/
```

## 目录结构

```
ocvault/
├── raw/                    # 原始 PDF 文件
├── knowledge/              # Obsidian 知识库
│   ├── 00-Meta/           # 索引文件
│   ├── 01-发行条款/       # 债券发行条款
│   ├── 02-募集资金运用/   # 资金用途
│   ├── 03-发行人基本情况/ # 发行人信息
│   ├── 04-财务状况/       # 财务分析
│   ├── 05-区域分析/       # 区域经济财政
│   └── 06-模板参考/       # Obsidian 模板
└── scripts/               # 处理脚本
```

## 提取字段说明

### 第二节 发行条款
- 发行人全称
- 债券全称/简称
- 注册规模
- 本期发行规模
- 债券期限
- 增信措施

### 第三节 募集资金运用
- 募集资金总额
- 资金使用计划
- 项目建设类（合法性文件）
- 偿还贷款类

### 第四节 发行人基本情况
- 发行人概况
- 历史沿革
- 股权结构
- 重要权益投资
- 主营业务（收入、毛利、毛利率）

### 第五节 财务状况
- 主要财务数据
- 经营性/非经营性资产
- 政府性资产识别
- 偿债能力指标

## 标签体系

- `#债券类型`: `#公司债` `#乡村振兴债` `#革命老区债`
- `#增信方式`: `#保证担保` `#抵押担保` `#质押担保` `#信用`
- `#行政层级`: `#省级` `#地市级` `#区县级`

## Obsidian 插件推荐

1. **Dataview** - 结构化数据查询
2. **Templates** - 统一笔记格式
3. **Tag Wrangler** - 标签管理

## 批量处理

```bash
# 处理 raw/ 目录下所有 PDF
python3 scripts/extract_prospectus.py

# 查看生成的笔记数量
find knowledge -name "*.md" | wc -l
```

## 注意事项

1. **PDF 格式**：确保 PDF 为文本可选格式（非扫描图片）
2. **数据校验**：提取的数据需要人工校验，特别是财务数字
3. **增量更新**：新 PDF 放入 raw/ 后重新运行脚本即可
4. **Git 备份**：已配置 git，可定期 commit 备份

## 常见问题

**Q: 提取的数据不准确？**
A: PDF 格式差异可能导致提取偏差，建议人工核对关键字段

**Q: 如何修改模板？**
A: 编辑 `knowledge/01-发行条款/_template.md` 等模板文件

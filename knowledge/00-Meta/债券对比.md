# 债券对比

## 条款对比

```dataview
TABLE WITHOUT ID
    bond_short AS "债券简称",
    issuer AS "发行人",
    year AS "年份",
    guarantee AS "增信措施",
    credit_rating AS "主体评级",
    bond_rating AS "债项评级",
    file.link AS "详情"
FROM "01-发行条款"
WHERE type = "bond_terms"
SORT issuer ASC
```

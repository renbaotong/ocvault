# 债券发行索引

```dataview
TABLE without id 
  file.link as "债券简称",
  type as "类型",
  tags as "标签"
FROM "01-发行条款"
WHERE type = "bond_terms"
SORT file.name ASC
```

## 统计

```dataviewjs
const bonds = dv.pages('"01-发行条款"').where(p => p.type == "bond_terms");
dv.table(
  ["类型", "数量"],
  Object.entries(
    bonds.groupBy(b => b.tags?.filter(t => t.startsWith('#'))?.join(', ') || '未分类')
      .map(g => [g.key, g.rows.length])
  ).sort((a, b) => b[1] - a[1])
);
```

# 债券索引

## 按债券类型统计

```dataview
TABLE
    length(rows) AS "数量",
    join(rows.bond_short, "、") AS "债券简称"
FROM "01-发行条款"
WHERE type = "bond_terms"
GROUP BY bond_type AS "债券类型"
SORT bond_type ASC
```

## 按增信方式统计

```dataview
TABLE
    length(rows) AS "数量"
FROM "01-发行条款"
WHERE type = "bond_terms"
GROUP BY guarantee AS "增信方式"
SORT length(rows) DESC
```

## 索引表

```dataview
TABLE WITHOUT ID
    bond_short AS "债券简称",
    issuer AS "发行人",
    year AS "年份",
    guarantee AS "增信方式",
    credit_rating AS "主体评级",
    bond_rating AS "债项评级",
    file.link AS "发行条款"
FROM "01-发行条款"
WHERE type = "bond_terms"
SORT bond_short ASC
```

## 评级分布

```dataview
TABLE
    length(rows) AS "数量",
    join(rows.bond_short, "、") AS "债券"
FROM "01-发行条款"
WHERE type = "bond_terms" AND credit_rating
GROUP BY credit_rating AS "主体评级"
SORT credit_rating DESC
```

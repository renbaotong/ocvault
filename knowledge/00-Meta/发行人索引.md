# 发行人索引

## 全部发行人

```dataview
TABLE WITHOUT ID
    issuer AS "发行人",
    bond_type AS "债券类型",
    file.link AS "概况"
FROM "03-发行人基本情况"
WHERE type = "issuer_profile"
SORT issuer ASC
```

## 按债券类型统计

```dataview
TABLE
    length(rows) AS "数量",
    join(rows.issuer, "、") AS "发行人"
FROM "03-发行人基本情况"
WHERE type = "issuer_profile"
GROUP BY bond_type AS "债券类型"
SORT bond_type ASC
```

## 按区域统计

```dataview
TABLE
    length(rows) AS "数量",
    join(rows.issuer, "、") AS "发行人"
FROM "03-发行人基本情况"
WHERE type = "issuer_profile"
GROUP BY regexreplace(issuer, "^(..).*", "$1") AS "区域"
SORT 区域 ASC
```

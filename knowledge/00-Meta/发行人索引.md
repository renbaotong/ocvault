# 发行人索引

```dataview
TABLE without id
  file.link as "发行人",
  type as "类型"
FROM "03-发行人基本情况"
WHERE type = "issuer_profile"
SORT file.name ASC
```

## 按区域统计

```dataviewjs
const issuers = dv.pages('"03-发行人基本情况"').where(p => p.type == "issuer_profile");
dv.table(
  ["省份", "数量"],
  Object.entries(
    issuers.groupBy(b => b.province || '未分类')
      .map(g => [g.key, g.rows.length])
  )
);
```

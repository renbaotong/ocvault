# 评级汇总

## 主体评级分布

```dataview
TABLE
    length(rows) AS "数量",
    join(rows.bond_short, "、") AS "债券"
FROM "01-发行条款"
WHERE type = "bond_terms" AND credit_rating
GROUP BY credit_rating AS "主体评级"
SORT credit_rating DESC
```

## 债项评级分布

```dataview
TABLE
    length(rows) AS "数量",
    join(rows.bond_short, "、") AS "债券"
FROM "01-发行条款"
WHERE type = "bond_terms" AND bond_rating
GROUP BY bond_rating AS "债项评级"
SORT bond_rating DESC
```

## 增信方式分布

```dataview
TABLE
    length(rows) AS "数量",
    join(rows.bond_short, "、") AS "债券"
FROM "01-发行条款"
WHERE type = "bond_terms"
GROUP BY guarantee AS "增信方式"
SORT length(rows) DESC
```

## 无增信债券列表

```dataview
TABLE WITHOUT ID
    bond_short AS "债券简称",
    issuer AS "发行人",
    credit_rating AS "主体评级",
    file.link AS "条款"
FROM "01-发行条款"
WHERE type = "bond_terms" AND (guarantee = "信用" OR guarantee = "无担保")
SORT issuer ASC
```

## 担保债券列表

```dataview
TABLE WITHOUT ID
    bond_short AS "债券简称",
    issuer AS "发行人",
    guarantee AS "担保方",
    credit_rating AS "主体评级",
    bond_rating AS "债项评级",
    file.link AS "条款"
FROM "01-发行条款"
WHERE type = "bond_terms" AND guarantee != "信用" AND guarantee != "无担保"
SORT issuer ASC
```

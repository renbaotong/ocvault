# 山东沂蒙兴业产业投资集团有限公司 - 募集资金运用

## 本期发行规模

- 发行规模：3.0 亿元

## 募集资金使用计划

本期债券募集资金3.0 亿元，扣除发行费用后，0.97亿元用于水利工程项目建设，0.84亿元用于蓝莓标准化种植，0.74亿元用于番茄种植，0.2亿元用于项目建设，0.16亿元用于投资项目的金额不超过募投项目资金缺口，0.08亿元用于项目部分的利息。

## 具体使用明细

| 用途 | 金额（亿元） | 占比 |
|-----|------------|------|
| 水利工程项目建设 | 0.97 亿元 | 32.4% |
| 蓝莓标准化种植 | 0.84 亿元 | 28.1% |
| 番茄种植 | 0.74 亿元 | 24.7% |
| 项目建设 | 0.2 亿元 | 6.7% |
| 投资项目的金额不超过募投项目资金缺口 | 0.16 亿元 | 5.4% |
| 项目部分的利息 | 0.08 亿元 | 2.7% |
| **合计** | 2.99亿元 | 100% |

# 山东阳都智圣产业投资集团有限公司 - 募集资金运用

## 本期发行规模

- 发行规模：3.00 亿元

## 募集资金使用计划

| 用途 | 金额（亿元） | 占比 |
|-----|------------|------|
| 项目建设 | 3.00 亿元 | 100.0% |
| 补充流动资金 | / | 0.0% |
| **合计** | 3.0亿元 | 100% |

## 资金用途概述

本期债券募集资金扣除发行费用后，全部用于革命老区振兴发展项目建设，包括阳都城市客厅项目和沂南县城乡供水一体化二期工程。

## 项目使用情况

### 项目建设类

| 项目名称 | 实施主体 | 项目总投资（亿元） | 募集资金使用额度（亿元） | 占项目总投资比例 | 项目建设内容 |
|---------|---------|------------------|----------------------|----------------|-------------|
| 阳都城市客厅项目 | 沂南县阳都水务集团有限公司 | 6.88 | 1.35 | 19.62% | 建设城发大厦、青少年活动中心及城市剧场、酿酒博物馆、袁家大院历史展馆、戏剧博物馆、界湖商号、商业街以及停车场（位）等配套设施 |
| 沂南县城乡供水一体化二期工程 | 沂南县阳都水务集团有限公司 | 2.49 | 1.65 | 66.27% | 主要建设内容：1、村内户表改造；2、张庄水厂、高湖水厂、水厂扩建工程；3、农村饮水安全信息化系统建设，增设监测设施 |
| **合计** | | | **3.00** | | |

### 项目收益情况

| 项目名称 | 收益实现方式 |
|---------|-------------|
| 阳都城市客厅项目 | 项目主要营业收入来源于门票收入、青少年活动中心及城市剧场收入、租赁收入、车位收入、物业收入 |
| 沂南县城乡供水一体化二期工程 | 收入来源于供水收入、管网租赁收入等 |

### 偿还项目贷款类

本期债券募集资金全部用于项目建设，不涉及偿还存量债务。

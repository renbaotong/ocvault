# 石家庄市鹿泉区产业发展投资有限公司 - 募集资金运用

## 本期发行规模

- 发行规模：5.0 亿元

## 募集资金使用计划

本期债券募集资金5.0 亿元，扣除发行费用后，1.78亿元用于发行人及子公司以合同方式有偿开展工程、取得货物或服务等行为，1.14亿元用于科技创新领域投资，0.83亿元用于投资基金合伙企业（有限合伙）、石家庄市共，0.8亿元用于科技创新领域投资，0.34亿元用于偿还公司到期有息负债，0.11亿元用于投资的比例（不低于发行规模的70%）或用。

## 具体使用明细

| 用途 | 金额（亿元） | 占比 |
|-----|------------|------|
| 发行人及子公司以合同方式有偿开展工程、取得货物或服务等行为 | 1.78 亿元 | 35.6% |
| 科技创新领域投资 | 1.14 亿元 | 22.8% |
| 投资基金合伙企业（有限合伙）、石家庄市共 | 0.83 亿元 | 16.6% |
| 科技创新领域投资 | 0.8 亿元 | 16.0% |
| 偿还公司到期有息负债 | 0.34 亿元 | 6.8% |
| 投资的比例（不低于发行规模的70%）或用 | 0.11 亿元 | 2.2% |
| **合计** | 5.00亿元 | 100% |

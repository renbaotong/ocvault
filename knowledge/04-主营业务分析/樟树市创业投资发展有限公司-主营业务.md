# 樟树市创业投资发展有限公司 - 主营业务分析

## 业务概况

樟树市创业投资发展有限公司是樟树市人民政府下属重要的国有资产运营和投资平台。公司成立于2022年，承担了樟树市供水、砂石资源开发、产业园区运营等重要公共事业和经营性业务。公司通过整合江西樟帮实业发展有限公司、樟树市供水有限公司、樟树市赣袁砂石有限公司等子公司，形成了多元化的业务布局。

## 主要业务板块

### 1. 供水业务

**业务内容**：
由子公司樟树市药都供水有限公司负责，主要从事自来水生产与供应、天然水收集与分配、污水处理及其再生利用等业务。

**经营情况**：
- 供水区域覆盖樟树市主要城区及周边乡镇
- 拥有第二水厂、第四水厂等供水设施

### 2. 砂石销售业务

**业务内容**：
由子公司樟树市赣袁砂石有限公司负责，主要从事河道采砂、矿产资源开采、建筑材料销售等业务。

**经营情况**：
- 注册资本26,000万元
- 是樟树市重要的砂石资源开发运营主体
- 为区域内建筑行业提供原材料供应

### 3. 产业园区运营

**业务内容**：
由子公司江西樟帮实业发展有限公司负责，主要从事项目投资、园区管理服务、物业管理、会议会展服务等业务。

**经营情况**：
- 注册资本10,000万元
- 负责樟树市相关产业园区的投资建设和运营管理

### 4. 保障性住房投资

**业务内容**：
由子公司樟树市城市保障住房投资有限公司负责，主要从事保障房投资、城镇化建设、房屋经营管理等业务。

**经营情况**：
- 成立于2012年，注册资本5,000万元
- 承担樟树市保障性住房建设运营职能

## 营业收入构成

| 业务板块 | 主要运营主体 | 业务性质 |
|---------|-------------|---------|
| 供水业务 | 樟树市药都供水有限公司 | 自来水生产与供应 |
| 砂石销售 | 樟树市赣袁砂石有限公司 | 矿产资源开采与销售 |
| 园区运营 | 江西樟帮实业发展有限公司 | 园区管理、项目投资 |
| 保障房投资 | 樟树市城市保障住房投资有限公司 | 保障房建设与运营 |
| 餐饮服务 | 樟树市国康餐饮服务有限公司 | 餐饮服务、食品销售 |

## 上下游产业链

### 上游供应商
- 水源供应方
- 砂石开采设备供应商
- 建筑施工材料供应商

### 下游客户
- 居民用水客户
- 工业用水企业
- 建筑施工企业
- 产业园区入驻企业

## 行业地位与竞争优势

### 行业地位
- 樟树市重要的国有资产运营主体
- 樟树市主要的供水服务提供商
- 区域内重要的砂石资源开发企业

### 竞争优势
1. **政府支持**：作为市属直管企业，获得樟树市政府的大力支持
2. **资源整合**：整合了供水、砂石、园区等多个优质资产
3. **区域专营**：供水业务具有区域专营性
4. **产业协同**：各业务板块之间形成良好的协同效应
